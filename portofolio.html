About Me

Nama Lengkap Saya Ahmad Ulul Azmi, Saya lahir Jakarta, 06 Januari 1992, Panggilan saya Ulul atau Azmi Atau Ahmad. intinya Saya suka cara orang memanggil saya dengan banyak alasan

Saya merupakan anak pertama dari 3 bersaudara, awal ketertarikan diri saya masuk ke dalam dunia IT karena keingin tahuan saya dalam membuat game bahasa indahnya adalah bermain game saat sekolah dasar. Tetapi karena keterbatasan informasi, ilmu dan banyak faktor semuah hal tentang game dan cara create game membuat diri saya lebih memilih untuk mengikuti arus anak - anak sewajarnya. Sehingga pada awal pertama kali saya lulus SMA disini peluang saya memilih kehidupan yang ingin saya jalani. Yaitu bagaimana membuat program kecil yang dapat saya nikmati sendiri
